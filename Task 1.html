ARDUINO

Arduino is an open-source hardware and software company, project and user community that designs and manufactures single-board microcontrollers and
microcontroller kits for building digital devices.

A program for Arduino hardware may be written in any programming language with compilers that produce binary machine code for the target processor.
Atmel provides a development environment for their 8-bit AVR and 32-bit ARM Cortex-M based microcontrollers: AVR Studio (older) and Atmel Studio
